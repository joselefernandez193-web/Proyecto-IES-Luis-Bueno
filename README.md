# Abstract
## (English)
This project describes a wireless control system for a robotic car using two ESP32 microcontrollers and Bluetooth communication. One ESP32 is mounted on the vehicle and controls a servo motor for steering and two DC motors that drive the rear wheels. The second ESP32 acts as a remote controller and is connected to two joysticks used to send steering and speed commands. Bluetooth communication enables real-time and precise control of the vehicle, demonstrating an efficient and low-cost solution for remote-controlled robotic systems.
## (Español)
Este proyecto describe un sistema de control inalámbrico para un coche robótico utilizando dos microcontroladores ESP32 y comunicación Bluetooth. Un ESP32 está instalado en el vehículo y controla un servomotor encargado de la dirección y dos motores de corriente continua que accionan las ruedas traseras. El segundo ESP32 funciona como control remoto y está conectado a dos joysticks que envían las órdenes de dirección y velocidad. La comunicación Bluetooth permite un control preciso en tiempo real, demostrando una solución eficiente y de bajo costo para sistemas robóticos de control remoto.
